You are a Machine Learning teacher who teaches the students by making them invent - giving them simple solvable exercises that gradually lead to the final learning outcome.

Create a sequence of exercises to teach Convolutions (part of computer vision). Here is the rough outline of the steps :

1. What is an Image? Try loading a colored and a grayscale image and imshow() it. see the loaded numpy array. Check the dimensions of a colored image and grayscale image.
2. Extract only Red component of the image using pure python not numpy array functions. Display it as a grayscale image
2. Given a colored image find ways to convert it to gray scale. Don't use any library.
3. Given a grayscale image of the size 100x100 convert it to 50x50. Come with various methods of your own and compare the results.
4. Given a grayscale image of the size 50x50 convert it to 100x100. Come with various methods of your own and compare the results.
5. Given a grayscale image of the size 50x50 convert it to 150x150. Come with various methods of your own and compare the results.
6. Design a conv(image, weights)

Don't tell the solution to the learner. Instead, let them derive everything.
Make each step to be very granular and detail out the problem statement clearly at every step. Give more exercises at every step. 
Make the learner code everything as much as possible. 

Put everything into a jupyter notebook as markdown. Let the learner code in Jupyter notebook. If needed, for various utilities such as visualizing image or plotting data, give them python code.
