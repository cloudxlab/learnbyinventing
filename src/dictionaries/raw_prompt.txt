You are a Python teacher who teaches students by making them invent — giving them simple solvable exercises that gradually lead to the final learning outcome.

Create a sequence of exercises to teach Python dictionaries. Here is the rough outline, adapted from an earlier in-person course (source: course_repos/june_2025_iitr/Exercises/6. Dictionaries.md):

PART 0 — DICT BASICS (warm-up, not in source)
- safe_lookup(d, key): return value or "not found" using dict.get
- count_items(items): count occurrences of each item in a list using a dict
  (This builds the core loop pattern: check-if-exists, update-or-create)

PART 1 — WORD COUNT
- word_count(text): split text, count word frequencies (reuses count_items)
- most_frequent_word(text): find word with highest count
- top_n_words(text, n): return top n (word, count) tuples sorted by frequency

PART 2 — ANAGRAMS
- anagram_key(word): sort letters into canonical string — "listen" → "eilnst"
- find_anagrams(text): group words by their anagram_key
- find_true_anagram_groups(text): only groups with 2+ words (filter find_anagrams)

PART 3 — EXPENSE TRACKER
- update_expenses(expense_list, expenses_dict): accumulate (category, amount) tuples in place
- print_expenses(expenses_dict): print "category : amount" per line
- top_expense(expenses_dict): return (category, amount) with highest spend
- over_budget(expenses_dict, limits): return {category: (spent, limit)} for overspent categories

BONUS
- invert_dict(d): swap keys and values
- group_by(items, key_func): general grouping — takes a list and a function, groups by key_func result
- find_anagrams_v2(text): rewrite find_anagrams as a one-liner using group_by + anagram_key

Key pedagogical arc:
1. The core dict pattern (check-if-exists → update-or-create) appears in count_items,
   find_anagrams, and update_expenses — learners discover the pattern independently in each context.
2. group_by is the generalisation that subsumes all three: collect items into buckets by a key function.
3. find_anagrams_v2 at the end shows that the complex-looking anagram exercise collapses to one line
   once you have the right abstraction.
