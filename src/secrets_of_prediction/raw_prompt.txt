Source exercises:
- Apples & parking story: 3 apples -> 6, 5 apples -> 8; predict 10 apples (13).
  y = b0 + b1*x, two equations, solve by hand.
- Q: come up with your dot_product for computing similarity. dot_product([1,2],[3,4]) = 11
- Q: convert a vector to unit vector by dividing each value by its length
  convert_to_unit([3,4]) -> [0.6, 0.8]
- Apr 19: cosine similarity via unit vectors (cos(A-B) = x1*x2 + y1*y2)
- Q: Matrix multiplication matrix_mul(A, B)
- Solve Equations recursively (eliminate one variable -> n-1 -> ... -> 1), findz
  back-substitution. NOTE: full solver already built in Loops and Arrays Part 5 —
  this chapter has the learner paste it in and use it, not rebuild it.
- Complexity: O(n^3); 4 vars 1000ms -> 8 vars ~8000ms quiz.
- Traffic quadratic: t=1->6, 2->12, 3->20, traffic = a*t^2+b*t+c, predict t=4 (30).
- solve_poly(x, y, xp): n points -> degree n-1 polynomial -> evaluate at xp
  solve_poly([1,2,3],[6,12,20],4) = 30
- Factory: (1,8),(2,11),(3,16) — no line fits. E(m,c) = sum (mx+c-y)^2;
  derive normal equations (source: 28m+12c=156, 12m+6c=70, i.e. 14m+6c=78, 6m+3c=35)
  -> m=4, c=11/3. Generalize to best_line(data) via sums. Plot provided.

Pedagogical arc: vectors (dot product -> unit -> cosine sim -> matmul) ->
prediction = equations + solving -> exact polynomial fitting -> least squares
(Ordinary Least Squares). Closing: O(n^3) + no-formula-derivatives motivate the
Gradient Descent chapter.
