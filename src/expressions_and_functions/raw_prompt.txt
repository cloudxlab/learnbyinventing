You are a Python teacher who teaches students by making them invent — giving them simple solvable exercises that gradually lead to the final learning outcome.

Create a sequence of exercises to teach expressions and functions. Here is the rough outline of the steps, adapted from an earlier in-person course (source: course_repos/june_2025_iitr/Exercises/1. Expressions and Functions.md):

1. Extract the last digit from a number, using the `%` (modulo) operator.

2. Remove the last digit from a number, using the `//` (integer division) operator.

3. Calculate Simple Interest:
   Interest = (Principal * Rate * Time) / 100
   calculate_interest(1000, 5, 2)    -> 100.0
   calculate_interest(1500, 4.3, 3)  -> 193.5
   calculate_interest(500, 10, 0)    -> 0.0

4. Calculate the Hypotenuse of a right triangle using the Pythagorean theorem:
   hypotenuse = (base**2 + height**2) ** 0.5
   calculate_hypotenuse(3, 4)   -> 5.0
   calculate_hypotenuse(5, 12)  -> 13.0

5. Find the distance between two points in 2D, generalizing the hypotenuse formula:
   find_distance_2d(0, 0, 3, 4)  -> 5.0
   find_distance_2d(1, 2, 4, 6)  -> 5.0

6. Extend distance to 3D:
   calculate_distance_3d(0, 0, 0, 1, 1, 1)  -> 1.7320508075688772
   calculate_distance_3d(1, 2, 3, 4, 6, 9)  -> 7.810249675906654

7. Find the Manhattan distance (sum of absolute differences along each axis, as opposed to the straight-line/Euclidean distance) and compare it with the 2D distance above:
   manhattan_distance(2, 3, 5, 1)    -> 5
   manhattan_distance(-1, -1, 1, 1)  -> 4

8. Predict y on a straight line y = m*x + c, given m, c, x:
   predict(2, 3, 4)   -> 11
   predict(-1, 5, 2)  -> 3

9. Given two points, write fit(x1, y1, x2, y2) to learn the slope and intercept of the line through them:
   fit(1, 2, 3, 6)  -> (2.0, 0.0)
   fit(0, 5, 2, 9)  -> (2.0, 5.0)

10. Find the first derivative of a single-argument function using the forward-difference method:
    f'(x) ≈ (f(x + h) - f(x)) / h
    approx_derivative(square, 3, 0.0001)  -> approx 6


Don't tell the solution to the learner. Instead, let them derive everything.
Make each step to be very granular and detail out the problem statement clearly at every step. Give more exercises at every step.
Make the learner code everything as much as possible.

Put everything into a jupyter notebook as markdown. Let the learner code in Jupyter notebook. For plotting give them python function to plot in Jupyter.

This chapter is a prerequisite for gradient_descent: predict()/fit() (line equation) and approx_derivative() (numerical derivative) are the building blocks that chapter re-derives and turns into an automatic search for the minimum of a function.
