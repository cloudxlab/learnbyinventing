RAG & Agentic AI chapter — ported verbatim from
jan2026_ml_dl/33_rag_agentic_md/, where the author had already written a
complete Learn-By-Inventing-style curriculum. Not generated; copied.

Files:
- rag_and_agents.md   <- rag_agentic_curriculum.md (unchanged)
- acme_datasets.py    <- companion datasets (DOCUMENTS, CHUNKS, MINI_CORPUS,
                         QA_EVAL_SET, STRUCTURED_DOCUMENTS...; no external libs)
- provider_setup.md   <- Anthropic/OpenAI setup + provider-agnostic wrapper

Structure (5 modules, ~14 exercises, each with Observe / The problem /
Your task / Reveal):
1. The Knowledge Gap — what LLMs know, hallucination, context stuffing
2. The Search Problem — keyword search, embeddings, semantic search
3. Inventing RAG — chunking, retrieval + generation, evaluation
4. The Single-Step Limit — where one-shot RAG fails
5. Inventing Agents — tool use, loops, multi-step reasoning; Epilogue

Unlike other chapters this is markdown, not a notebook (exercises require an
LLM API key and are run in the learner's own environment). Rendered to HTML
via pandoc in build.sh (both rag_and_agents.md and provider_setup.md); the
source repo's learning_by_inventing.html render was NOT copied — we generate
our own to match the site.
