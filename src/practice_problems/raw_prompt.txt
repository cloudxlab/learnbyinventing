47 standalone ML-themed drill notebooks organized by Python concept:
  01_expressions (9): mean, z-score, min-max, euclidean, independent prob,
    squared error, sigmoid, covariance term, gradient step
  02_if_else (9): outlier flag, prob classify, sigmoid→class, confusion cell,
    nearest centroid, correlation strength, safe normalize, train/val/test
    split by position, pure node
  03_loops (10): mean, sum sq dev, std dev, outliers, min-max list, kNN
    distances, confusion matrix, covariance, gradient descent, entropy
  04_functions (10): mean/variance/std, outliers, kNN predict, precision/
    recall/F1, correlation, regression pipeline, entropy+info gain,
    sigmoid+perceptron
  05_recursion (9): sum, product, sum sq dev, factorial, n-choose-k, binary
    search, merge sort, LR decay, tree depth

Format: problem statement + starter cell with # TODO + test cell that
prints "All tests passed!" or raises AssertionError. Self-contained; no
guided arc. Notebooks copied as-is (no LBI-style conversion needed).

Design decisions:
- Not converted to LBI guided-invention format — these are drill exercises,
  a different learning mode (apply a known formula vs. derive from scratch).
- Markdown index chapter (practice_problems.md) explains the difference and
  groups by LBI chapter cross-reference. Pandoc-exported; individual notebooks
  accessed via Colab links (no nbconvert loop needed).
- Concepts all appear in LBI chapters: stats in loops_and_array, gradient
  descent in gradient_descent, entropy in information_theory, neural net
  perceptron in neural_networks, tree depth in trees_and_heaps.
- "Best done after" annotations in the index tell learners when to revisit.
