Source exercises:
- Wild card character matcher: * matches 0 or more chars, ? matches exactly one char.
    a* -> abc, ab, ax, a
    a? -> a1, a2, aa
    a*b -> axyzb, a123b, ab
    a*b*c -> abc, abbbc, a1b1c
    aa?b*: match aa1b, aaxby, aa1bcdeffgshshshsh; not aab, ab
  def ismatch(pat, text): return True/False
- Simple pattern matcher: . matches single char, * matches any number of any chars.
  Recursive strategy sketch given in the source (try text[1:], text[2:], ...).
- Regex table (^ $ . \s \S * *? + +? [aeiou] [^XYZ] [a-z0-9] parentheses).
- Date matching: [0-9][0-9][-/][0-9][0-9][/-][0-9]{4}; stricter day 0[1-9]|[12][0-9]|30|31
  and month 0[1-9]|1[0-2]. Valid/invalid date test lines in source.
- Credit card numbers with dashes/spaces/nothing.
- Q1: regex for email address. Q2: URL. Q3: extract URLs from server logs access.log.41.
